# Football Player Recommendation System

Sistem rekomendasi pemain sepak bola berbasis ontologi dan NLP yang dibangun dengan Python, Flask, dan spaCy.

## Fitur

- **Pencarian Natural Language**: Cari pemain menggunakan bahasa natural (Inggris)
- **Rekomendasi Otomatis**: Rekomendasi berdasarkan kategori (Young Talents, Best Value, Top Scorers)
- **Analisis Statistik**: Dashboard statistik database pemain
- **Detail Pemain Lengkap**: Informasi detail setiap pemain dengan analisis value
- **Responsive Design**: Antarmuka yang responsif dan user-friendly

## Teknologi yang Digunakan

- **Backend**: Python, Flask, RDFLib
- **NLP**: spaCy, NLTK
- **Ontologi**: RDF/OWL (Protégé)
- **Frontend**: HTML, CSS, JavaScript (Vanilla)
- **Data Processing**: Jupyter Notebook, pandas
